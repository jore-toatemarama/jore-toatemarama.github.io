What is the difference between Margin, Border, and Padding?

To put it simply using the CSS Box Model, Margin, Border and Padding are used to define where and how you want something on the webpage. you use pixels as a unit to increase or decrease top bottom left and right directions for either margin or padding.

Border

Border is fairly self explanatory, editing these variables changes the border of the element you are interacting with. you can change the color of the border, the size, width pattern and spacing. You can even specify just one side to have the border on. this is the middle layer, it sits between the margin on the outside and the padding on the inside.

Margin

Margin is the space between the border and the edge of element. this can be manipulated to push the element away from the edges of the page, or off of other elements in order to create distance and symmetry on the page. the margin is important in shaping how the overall page looks, and can be edited using clockwise notation in order to have more control. Negative margins can also be used for control or to overlap elements

Padding

Padding is a bit more complicated as it takes up the space used bewtween the border and the content. this means increasing the padding can either decrease the size of the content or increase the size of the element itself in order to fit, by default it will increase the element, padding is useful for making the clickable space on buttons bigger, or for creating whitespace inside the element between the content and border.
